# ERPHouse

Sistema ERP simples para controle de produtos, entradas e saídas,
desenvolvido em Python com SQLite.

## Funcionalidades atuais
- Cadastro de produtos
- Controle de usuários (humano / sistema)
- Registro de movimentações
- Atualização automática de estoque

## Tecnologias
- Python
- SQLite

## Objetivo
Projeto de portfólio com foco em organização de backend e boas práticas.
